# Web Scraper
A web scraper application.

## Run Locally
Clone the project

```bash
  git clone https://github.com/kodegrenade/web-scraper
```

Go to the project directory

```bash
  cd web-scraper
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

**Check your console for scrapped data**
